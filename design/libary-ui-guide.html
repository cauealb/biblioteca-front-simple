Roadmap React

Ordem boa para implementar sem se perder.

Cada passo adiciona um conceito novo, mas aproveita o que voce ja fez.

01

Busca melhorada

Use includes, estado do input, contador de resultados e estado vazio.

02

Cadastro de livro

Crie BookForm com campos controlados e adicione livros com setBooks.

03

Modal de detalhes

Guarde o livro selecionado em selectedBook e mostre detalhes em um modal.

04

Editar, excluir e filtrar

Pratique map, filter, confirmacao e status por abas.

05

localStorage

Salve os livros com useEffect para eles continuarem apos recarregar a pagina.

06

Context API + hook proprio

Crie BooksContext e depois um useBooks para compartilhar dados entre telas.

Regras de UI

Pequenas decisoes que deixam o app mais maduro.

Use estas regras como checklist quando terminar cada tela.

Estado vazio

Sempre mostre algo quando nao houver livros, resultados ou dados no dashboard.

Feedback

Ao criar, editar ou excluir, mostre uma mensagem simples de sucesso ou erro.

Status claro

Use cores fixas: verde para lido, amarelo para lendo e vermelho para pendente.

Componentes pequenos

Se um arquivo passar de muita responsabilidade, extraia componentes menores.

Tipos primeiro

Crie um tipo Book antes de implementar filtros, cards e formulario.

Mobile

Antes de considerar pronto, reduza a tela e veja se sidebar, cards e modal continuam usaveis.
